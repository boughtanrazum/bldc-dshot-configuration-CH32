## 📋 Описание

Приложение предназначено для упрощения настройки и генерации файлов конфигурации для проектов, использующих:

- **BLDC драйвер** на расширенных таймерах (TIM1, TIM8)
- **DSHOT протокол** на любых доступных таймерах

Программа автоматически проверяет конфликты таймеров и пинов, предупреждая пользователя о возможных проблемах перед генерацией.

## ✨ Возможности

- Выбор микроконтроллера: CH32V003, CH32V203
- Настройка расширенного таймера для BLDC с выбором remap
- Настройка таймера для DSHOT с выбором скорости и remap
-  Ручная/автоматическая настройка компараторных пинов
-  Включение быстрых прерываний для BLDC и DSHOT
-  Валидация конфигурации в реальном времени
-  Автоматическое обнаружение конфликтов:
  - Одинаковые таймеры для BLDC и DSHOT
  - Пересечения пинов DSHOT CH1 с компараторными пинами
  - Дублирование компараторных пинов
- 📁 Генерация файлов `config.h` и `config-2.h`


## 🚀 Установка и запуск

### Требования
- Godot Engine 4.2 или выше
- Операционная система: Windows, Linux, macOS

### Запуск из Godot
1. Клонируйте репозиторий
2. Откройте проект в Godot Engine 4.2+
3. Запустите сцену `main.tscn`

### Сборка исполняемого файла
```bash
# Экспорт в Windows
godot --export "Windows Desktop" config_generator.exe

# Экспорт в Linux
godot --export "Linux/X11" config_generator

# Экспорт в macOS
godot --export "macOS" config_generator.app
